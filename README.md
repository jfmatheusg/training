# RED BICICLETAS

## Pre-requisitos

* NodeJS
* NPM

## Correr el proyecto

En la carpeta _red-bicicletas_ correr los siguientes comandos:

```
npm install
npm run start-dev
```

## Visualizar proyecto

En un navegador web acceder a la aplicación

```
localhost:3000
```

## Probar las APIs

Cargar la colección de Postman que se encuentra en _test_ y consumir las APIs
